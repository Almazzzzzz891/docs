## Авторизация
### **Запрос**
```
POST /login
{
    "email": <String>,
    "password": <String>
}
```

### **Ответ**
```json
{
    "message": <String>,
    "token": <String>
}
```

После этого полученный токен необходимо передавать или в [cookie](#cookie), или в [heaers](#headers)
